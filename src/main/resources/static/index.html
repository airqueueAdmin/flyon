최저가 결과

이용 방법

1. 항공권을 검색하고 가장 저렴한 항공편을 선택합니다.
2. 가격 추적을 누르고 필요하면 목표 가격을 입력합니다.
3. 카카오 알림을 켜고 로그인 연결을 완료하면 가격 하락이나 목표 가격 도달 시 알림을 받습니다.
4. 카카오 메시지의 버튼으로 추적 목록이나 스카이스캐너로 바로 이동합니다.
